з”ҹе‘Ҫе‘ЁжңҹеӨ§дҪ“жөҒзЁӢ

еҲқе§ӢеҢ–жӯҘйӘӨ

еҲқе§Ӣиҝӣе…ҘйЎөйқў вҶ’ constructor вҶ’ componentWillMount вҶ’ render вҶ’ componentDidMount вҶ’ componentWillUnmountпјӣ

дёӢйқўзҡ„еӣҫдёӯпјҢеҠ дәҶдёҖдәӣзҡ„зҠ¶жҖҒиҜҙжҳҺпјҢжҸҗеүҚиҜҙжҳҺдёҖдёӢпјҡ_compositeLifeCycleStateзҝ»иҜ‘дёәпјҡз»јеҗҲз”ҹе‘Ҫе‘ЁжңҹзҠ¶жҖҒ(дёӢж–Үз®Җз§°пјҡз”ҹе‘ҪзҠ¶жҖҒ)пјҢе®ғжҳҜз”ҹе‘Ҫе‘Ёжңҹдёӯе…ій”®зҡ„зҠ¶жҖҒпјҢsetStateжҳҜеҗҰеҮәеҸ‘жӣҙж–°пјҢе°ұжҳҜдҫқжҚ®е®ғзҡ„зҠ¶жҖҒ

1гҖҒйҰ–е…ҲпјҢиҝӣе…ҘйЎөйқўпјҢдјҡеҲқе§ӢеҢ–йЎөйқўж•°жҚ®(state, props, contextзӯүвҖҰ)пјҢзӯүеҫ…еӨҮз”Ёпјӣ

2гҖҒз„¶еҗҺпјҢи®ҫзҪ®з”ҹе‘ҪзҠ¶жҖҒдёәпјҡMOUNTINGпјҢиҝҷдёӘзҠ¶жҖҒдёӢйқўдјҡиҜҙжҳҺе®ғзҡ„з”ЁйҖ”пјҢиҝҷйҮҢжҲ‘д»¬е…ҲжҢүз…§жөҒзЁӢз»§з»ӯеҫҖдёӢиө°пјӣ

3гҖҒжҺҘдёӢжқҘпјҢеңЁcomponentWillMountдёӯпјҢsetStateж“ҚдҪңпјҢеҸӘжҳҜжҠҠstateеҗҲе№¶еҲ°еҲқе§ӢеҢ–зҠ¶жҖҒдёӯпјҢиҖҢж №жң¬дёҚдјҡи§ҰеҸ‘render пјӣеңЁиҝҷйҮҢжӣҙж–°stateпјҢе°ұзӯүеҗҢдәҺзӣҙжҺҘеҶҷеңЁthis.stateдёӯпјҢжүҖд»ҘпјҢеңЁжӯӨз”ҹе‘Ҫе‘Ёжңҹдёӯзҡ„setStateж №жң¬жІЎжңүж„Ҹд№үпјӣ

4гҖҒжү§иЎҢеҲ°иҝҷйҮҢпјҢз”ҹе‘ҪзҠ¶жҖҒдјҡиў«йҮҚзҪ®дёә nullпјҢд№ӢеҗҺжҳҜжёІжҹ“йЎөйқў(еҚіжү§иЎҢrender)пјӣ

5гҖҒжңҖеҗҺпјҢжёІжҹ“е®Ңд»ҘеҗҺпјҢжү§иЎҢcomponentDidMountпјҢиҝҷйҮҢдҪҝз”ЁsetStateеҚідјҡжӯЈеёёи§ҰеҸ‘йҮҚж–°жёІжҹ“дәҶпјҢжӣҙж–°stateгҖӮ(жҺҘдёӢжқҘпјҢе°ұжҳҜжӣҙж–°жөҒзЁӢдәҶпјҒпјҒ)

жӣҙж–°

setStateеҸ‘иө·жӣҙж–°жөҒзЁӢпјҢе№¶дёҚжҳҜжҜҸдёӘз”ҹе‘Ҫе‘Ёжңҹдёӯзҡ„setStateйғҪдјҡи§ҰеҸ‘и·ҹж–°пјҢе®ғеҲӨж–ӯзҡ„дҫқжҚ®е°ұжҳҜ _compositeLifeCycleState пјҢдёҠйқўиҜҙеҲ°зҡ„MOUNTINGе’ҢдёӢйқўе°Ҷдјҡи®ІеҲ°зҡ„RECEIVE_PROPSпјҢеңЁиҝҷдёӨдёӘзҠ¶жҖҒдёӢпјҢreactдјҡеӨ„зҗҶsetStateдёәдёҚдјҡи§ҰеҸ‘жӣҙж–°пјҢзӣёеҜ№зҡ„е…¶д»–зҠ¶жҖҒдјҡжӯЈеёёжӣҙж–°пјӣиҝҷд№ҹеҜјиҮҙдәҶдёҖдәӣз”ҹе‘Ҫе‘Ёжңҹдёӯзҡ„дҪҝз”Ёжғ…еҶө


1гҖҒйҰ–е…ҲпјҢreactдјҡжҜ”иҫғеүҚеҗҺе…ғзҙ гҖҒзҠ¶жҖҒзӯүжҳҜеҗҰдёҚеҗҢпјҢеҰӮжһңдёҚеҗҢеҲҷжӯЈејҸеҸ‘иө·жӣҙж–°пјӣ

2гҖҒз„¶еҗҺпјҢз”ҹе‘ҪзҠ¶жҖҒ иў«и®ҫзҪ®дёәRECEIVE_PROPS(жіЁж„ҸпјҡжӯӨж—¶з”ҹе‘Ҫе‘ЁжңҹдёӯпјҢsetStateдёҚдјҡи§ҰеҸ‘жӣҙж–°пјҢиҖҢжҳҜдјҡеҒҡе…¶д»–еӨ„зҗҶпјӣ

3гҖҒжҺҘдёӢжқҘпјҢcomponentWillReceivePropsдёӯзҡ„setStateе°ұдёҚдјҡжү§иЎҢжӣҙж–°пјҢиҖҢжҳҜеҗҲе№¶жҢӮиҪҪиө·жқҘпјҢзӯүеҫ…renderж—¶з»ҹдёҖжӣҙж–°пјӣ

4гҖҒеҲ°иҝҷйҮҢпјҢз”ҹе‘ҪзҠ¶жҖҒ дјҡйҮҚзҪ®дёәnullпјӣз„¶еҗҺshouldComponentUpdateдёӯдјҡеҲӨж–ӯжҳҜеҗҰжӣҙж–°пјӣд№ӢеҗҺжҳҜcomponentWillUpdateгҖӮ


жіЁж„ҸпјҡshouldComponentUpdateе’ҢcomponentWillUpdateжү§иЎҢзҡ„ж—¶еҖҷпјҢз”ҹе‘ҪзҠ¶жҖҒ е·Із»Ҹиў«йҮҚзҪ®дёәnullпјҢеңЁе®ғд»¬йҮҢйқўзҡ„setStateдјҡи§ҰеҸ‘жӣҙж–°пјҢйӮЈд№ҲеңЁе…¶й—ҙдҪҝз”Ёе‘ўпјҹдјҡйҖ жҲҗд»Җд№Ҳпјҹзӯ”жЎҲе°ұжҳҜпјҡеңЁдёҖдёӘжӣҙж–°е‘ЁжңҹиҝҳжІЎжңүrenderд№ӢеүҚпјҢеҶҚж¬ЎеҸ‘иө·updateComponentпјҢзӣҙжҺҘеҜјиҮҙйҖ’еҪ’жӣҙж–°жӯ»еҫӘзҺҜпјҢйҖ жҲҗжөҸи§ҲеҷЁеҚЎжӯ»гҖӮжүҖд»ҘеңЁд»–д»¬йҮҢйқўрҹҡ«зҰҒжӯўрҹҡ«дҪҝз”ЁsetStateгҖӮ

5гҖҒжңҖеҗҺпјҢжёІжҹ“йЎөйқўпјӣеҶҚжү§иЎҢcomponentDidUpdateпјӣе®ғйҮҢйқўжү§иЎҢsetStateпјҢдјҡи§ҰеҸ‘жӣҙж–°пјҢдёҚеҗҢзҡ„жҳҜrenderе®ҢжҲҗд№ӢеҗҺеҶҚеҸ‘иө·зҡ„reRenderгҖӮиҷҪз„¶иҝҷе„ҝеҢәеҲ«дәҺдёҠйқўдёӨдёӘз”ҹе‘Ҫе‘ЁжңҹдёӯдҪҝз”Ёзҡ„жғ…еҶөпјҢдҪҶжҳҜдјҡдёҖйҒҚдёҖйҒҚзҡ„жӣҙж–°пјҢеҸҜд»Ҙдҫқйқ shouldComponentUpdateиҝӣиЎҢжҺ§еҲ¶

жңҖеҗҺе‘ўпјҢз®ҖеҚ•д»Ӣз»ҚдёҖдёӢпјҢйҖҖеҮәйЎөйқўзҡ„жөҒзЁӢгҖӮжӯӨжөҒзЁӢдёӯпјҢ йҰ–е…Ҳз”ҹе‘ҪзҠ¶жҖҒд№ҹдјҡиў«иөӢдәҲеҖјдёәUNMOUNTINGпјҢ з„¶еҗҺжү§иЎҢcomponentWillUnmountпјҢжңҖеҗҺз”ҹе‘ҪзҠ¶жҖҒйҮҚзҪ®дёәnullпјҢеҒҡеҚёиҪҪйЎөйқўз»„д»¶е’ҢзҠ¶жҖҒзӯүеӨ„зҗҶгҖӮйЎәдҫҝжҸҗдёҖдёӢпјҢеңЁcomponentWillUnmountдёӯдҪҝз”ЁsetStatпјҢеӣ дёәзӯүеҫ…зҡ„жҳҜйЎөйқўеҚёиҪҪпјҢжүҖд»Ҙж”№еҸҳstateжҳҜжІЎжңүж„Ҹд№үзҡ„гҖӮ

ејӮжӯҘиҺ·еҸ–ж•°жҚ®

componentWillMountзҡ„й—®йўҳ

еңЁcomponentWillMountдёӯжү§иЎҢthis.setStateжҳҜдёҚдјҡи§ҰеҸ‘дәҢж¬ЎжёІжҹ“зҡ„гҖӮ

е®ғд№ҹеҸӘдјҡеңЁжҢӮиҪҪиҝҮзЁӢдёӯиў«и°ғз”ЁдёҖж¬ЎпјҢе®ғзҡ„дҪңз”Ёе’ҢconstructorжІЎжңүеӨӘеӨ§е·®ејӮгҖӮжңүеҫҲеӨҡдәәеңЁcomponentWillMountдёӯиҜ·жұӮеҗҺеҸ°ж•°жҚ®пјҢи®Өдёәиҝҷж ·еҸҜд»Ҙжӣҙж—©зҡ„еҫ—еҲ°ж•°жҚ®пјҢдҪҶжҳҜcomponentWillMoutжҳҜеңЁrenderеҮҪж•°жү§иЎҢеүҚжү§иЎҢзҡ„пјҢиҷҪз„¶иҜ·жұӮжҳҜеңЁз¬¬дёҖж¬Ўrenderд№ӢеүҚеҸ‘йҖҒзҡ„пјҢдҪҶжҳҜиҝ”еӣһе№¶дёҚиғҪдҝқиҜҒеңЁrenderд№ӢеүҚе®ҢжҲҗгҖӮrenderдёҚдјҡзӯүдҪ ж…ўж…ўиҜ·жұӮгҖӮжүҖд»ҘеҪ“ж•°жҚ®еҲ°жқҘж—¶з»„д»¶е·Із»Ҹе®ҢжҲҗйҰ–ж¬Ўзҡ„жёІжҹ“пјҢиҖҢеңЁиҝҷдёӘз”ҹе‘Ҫе‘ЁжңҹеҪ“дёӯsetStateеҸҲдёҚдјҡи§ҰеҸ‘reRenderпјҢжүҖд»ҘдјҡеҮәзҺ°иҜ·жұӮзҡ„ж•°жҚ®ж— жі•жӯЈеёёжёІжҹ“гҖӮ

componentdidmountзҡ„дјҳзӮ№

еңЁз”ҹдә§ж—¶,componentDidMountз”ҹе‘Ҫе‘ЁжңҹеҮҪж•°жҳҜжңҖеҘҪзҡ„ж—¶й—ҙеҺ»иҜ·жұӮж•°жҚ®,е…¶дёӯжңҖйҮҚиҰҒеҺҹеӣ :дҪҝз”ЁcomponentDidMountз¬¬дёҖдёӘеҘҪеӨ„е°ұжҳҜиҝҷдёӘдёҖе®ҡжҳҜеңЁз»„д»¶еҲқе§ӢеҢ–е®ҢжҲҗд№ӢеҗҺ,еҶҚдјҡиҜ·жұӮж•°жҚ®,еӣ жӯӨдёҚдјҡжҠҘд»Җд№ҲиӯҰе‘ҠжҲ–иҖ…й”ҷиҜҜ,жҲ‘д»¬жӯЈеёёиҜ·ж•ҷж•°жҚ®е®ҢжҲҗд№ӢеҗҺдёҖиҲ¬йғҪдјҡsetState.

дҪҝз”ЁжҖ»з»“

з”ҹе‘Ҫе‘ЁжңҹдёӯsetStateзҡ„дҪҝз”Ёжғ…еҶөпјҡ

  • ж— ж„Ҹд№үдҪҝз”ЁпјҡcomponentWillMountпјҢcomponentWillUnmountпјӣ

  • жңүжқЎд»¶дҪҝз”ЁпјҡcomponentDidUpdateпјӣ

  • зҰҒжӯўдҪҝз”ЁпјҡcomponentWillUpdateпјҢshouldComponentUpdateпјӣ

  • жӯЈеёёдҪҝз”ЁпјҡcomponentWIllReceivePropsпјҢcomponentDidMountгҖӮ

з”ҹе‘Ҫе‘ЁжңҹдёӯsetStateжҳҜеҗҰи§ҰеҸ‘жӣҙж–°пјҡ

componentWillMountе’ҢcomponentWillReceivePropsдёӯпјҢsetStateдјҡиў«reactеҶ…йғЁеӨ„зҗҶпјҢиҖҢдёҚи§ҰеҸ‘renderпјӣе…¶д»–з”ҹе‘Ҫе‘ЁжңҹеқҮжӯЈеёёеҮәеҸ‘жӣҙж–°жёІжҹ“гҖӮ

react V16.4зүҲжң¬з”ҹе‘Ҫе‘Ёжңҹ

еҺҹжқҘпјҲReact v16.0еүҚпјүзҡ„з”ҹе‘Ҫе‘ЁжңҹеңЁReact v16жҺЁеҮәзҡ„Fiberд№ӢеҗҺе°ұдёҚеҗҲйҖӮдәҶпјҢеӣ дёәеҰӮжһңиҰҒејҖеҗҜasync renderingпјҢеңЁrenderеҮҪж•°д№ӢеүҚзҡ„жүҖжңүеҮҪж•°пјҢйғҪжңүеҸҜиғҪиў«жү§иЎҢеӨҡж¬ЎгҖӮ

дёҠж–ҮжҲ‘д»¬е·Із»ҸзҹҘйҒ“пјҢдёӢйқўиҝҷдәӣз”ҹе‘Ҫе‘ЁжңҹжҳҜеңЁrenderд№ӢеүҚжү§иЎҢзҡ„пјҡ

  • componentWillMount
  • componentWillReceiveProps
  • shouldComponentUpdate
  • componentWillUpdate

react V16зүҲжң¬дҪҝз”Ёзҡ„fiberжһ¶жһ„дёӯпјҢе°ҶдёҠиҝ°з”ҹе‘Ҫе‘ЁжңҹеҪ’дёәдәҶеҗҢдёҖдёӘйҳ¶ж®өпјҡи°ғе’Ңйҳ¶ж®өгҖӮи°ғе’Ңйҳ¶ж®өзҡ„з”ҹе‘Ҫе‘ЁжңҹеңЁе®һйҷ…з»„д»¶зҡ„жёІжҹ“иҝҮзЁӢдёӯпјҢжңүдёҖе®ҡеҮ зҺҮдјҡеҸ‘з”ҹеӨҡж¬Ўи°ғз”ЁпјҢиҝҷеҸ–еҶідәҺfiberжһ¶жһ„еҪ“дёӯеҜ№дәҺж—¶й—ҙзүҮзҡ„жҺ§еҲ¶пјҢеҰӮжһңеңЁз”»йқўзҡ„1её§зҡ„ж—¶й—ҙд№ӢеҶ…дёҚиғҪеҝ«йҖҹе®ҢжҲҗиҝҗз®—пјҢз”ҹе‘Ҫе‘Ёжңҹдёӯжү§иЎҢзҡ„JSзәҝзЁӢдјҡиў«жү“ж–ӯпјҢдјҳе…ҲжёІжҹ“йЎөйқўйҳІжӯўйЎөйқўеҚЎйЎҝгҖӮиў«жү“ж–ӯзҡ„з”ҹе‘Ҫе‘ЁжңҹдјҡеңЁд№ӢеҗҺд»ҺеӨҙжү§иЎҢгҖӮ

зҰҒжӯўз”ЁжҜ”еҠқеҜјејҖеҸ‘иҖ…дёҚиҰҒиҝҷж ·з”Ёзҡ„ж•ҲжһңжӣҙеҘҪпјҢжүҖд»ҘйҷӨдәҶshouldComponentUpdateпјҢе…¶д»–еңЁrenderеҮҪж•°д№ӢеүҚзҡ„жүҖжңүеҮҪж•°пјҲcomponentWillMountпјҢcomponentWillReceivePropsпјҢcomponentWillUpdateпјүйғҪиў«getDerivedStateFromPropsжӣҝд»ЈгҖӮ

getDerivedStateFromProps

йҡҸзқҖgetDerivedStateFromPropsзҡ„жҺЁеҮәпјҢеҗҢж—¶deprecateдәҶдёҖз»„з”ҹе‘Ҫе‘ЁжңҹAPIпјҢеҢ…жӢ¬пјҡ

  • componentWillReceiveProps
  • componentWillMount
  • componentWillUpdate

д№ҹе°ұжҳҜз”ЁдёҖдёӘйқҷжҖҒеҮҪж•°getDerivedStateFromPropsжқҘеҸ–д»Јиў«deprecateзҡ„еҮ дёӘз”ҹе‘Ҫе‘ЁжңҹеҮҪж•°пјҢе°ұжҳҜејәеҲ¶ејҖеҸ‘иҖ…еңЁrenderд№ӢеүҚеҸӘеҒҡж— еүҜдҪңз”Ёзҡ„ж“ҚдҪңпјҢиҖҢдё”иғҪеҒҡзҡ„ж“ҚдҪңеұҖйҷҗеңЁж №жҚ®propsе’ҢstateеҶіе®ҡж–°зҡ„state

иҝҷдёӘgetDerivedStateFromPropsжҳҜдёҖдёӘйқҷжҖҒеҮҪж•°пјҢжүҖд»ҘеҮҪж•°дҪ“еҶ…дёҚиғҪи®ҝй—®thisпјҢз®ҖеҚ•иҜҙпјҢе°ұжҳҜеә”иҜҘдёҖдёӘзәҜеҮҪж•°пјҢзәҜеҮҪж•°жҳҜдёҖдёӘеҘҪдёңиҘҝе•ҠпјҢиҫ“еҮәе®Ңе…Ёз”ұиҫ“е…ҘеҶіе®ҡгҖӮ

getSnapshotBeforeUpdate

React v16.4иҝҳе…·жңүдәҶдёҖдёӘж–°зҡ„еЈ°жҳҺе‘ЁжңҹеҮҪж•°getSnapshotBeforeUpdateпјҢиҝҷеҮҪж•°дјҡеңЁrenderд№ӢеҗҺжү§иЎҢпјҢиҖҢжү§иЎҢд№Ӣж—¶DOMе…ғзҙ иҝҳжІЎжңүиў«жӣҙж–°пјҢз»ҷдәҶдёҖдёӘжңәдјҡеҺ»иҺ·еҸ–DOMдҝЎжҒҜпјҢи®Ўз®—еҫ—еҲ°дёҖдёӘsnapshotпјҢиҝҷдёӘsnapshotдјҡдҪңдёәcomponentDidUpdateзҡ„з¬¬дёүдёӘеҸӮж•°дј е…ҘгҖӮ

дёҠйқўиҝҷж®өд»Јз ҒеҸҜд»ҘзңӢеҮәжқҘиҝҷдёӘsnapshotжҖҺд№ҲдёӘз”Ёжі•пјҢsnapshotе’ӢзңӢиҝҳд»ҘдёәжҳҜз»„д»¶зә§еҲ«зҡ„жҹҗдёӘвҖңеҝ«з…§вҖқпјҢе…¶е®һеҸҜд»ҘжҳҜд»»дҪ•еҖјпјҢеҲ°еә•жҖҺд№Ҳз”Ёе®Ңе…ЁзңӢејҖеҸ‘иҖ…иҮӘе·ұпјҢgetSnapshotBeforeUpdateжҠҠsnapshotиҝ”еӣһпјҢз„¶еҗҺDOMж”№еҸҳпјҢз„¶еҗҺsnapshotдј йҖ’з»ҷcomponentDidUpdateгҖӮ
